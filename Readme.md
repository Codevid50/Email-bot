# Email Bot (Python CLI)

This is a simple command-line Email Bot built using Python.
It sends emails using Gmail SMTP and saves sent email details in a JSON file.

## Features
- Send emails from terminal
- Uses Gmail SMTP
- Stores email history in a JSON file
- Send multiple emails in one run

## Concepts Used
- smtplib
- EmailMessage
- JSON file handling
- Loops and conditions

## How to Run
1. Open the project folder
2. Run the program:
   python main.py
